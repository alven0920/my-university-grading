### Pre-requisite

* Node JS version >= 14.0.0 (NPM >= 6.14.4)

### Setup

1. Run `npm install` to install dependencies

### Technology Stack

1. TypeScript 4.x.x (JavaScript)
2. Jest (for unit testing)
3. eslint (for linting - format correction)
